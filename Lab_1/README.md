# Лабораторная работа №1: Основы Java

## Описание
Изучение базового синтаксиса Java, компиляция и работа с циклами.

## Файлы
* JavaHelloWorldProgram.java — классическая программа «Привет, мир».
* Primes.java — поиск простых чисел до 100.
* Palindrome.java — проверка строки на палиндром.

## Запуск
1. Компиляция: `javac ИмяФайла.java`
2. Запуск: `java ИмяКласса`
