# Лабораторная работа №4: Исключения в Java (Exception Handling)

## Описание
Работа посвящена изучению механизма обработки исключений в Java. Рассмотрены основные конструкции `try-catch-finally`, создание собственных классов исключений и работа с файловыми потоками с корректным закрытием ресурсов.

## Основные компоненты
* `ArrayAverage.java` — программа для вычисления среднего арифметического элементов массива с обработкой `NumberFormatException` и `ArrayIndexOutOfBoundsException`.
* `FileCopy.java` — утилита для копирования данных между файлами (`source.txt` -> `dest.txt`) с обработкой `FileNotFoundException` и `IOException` через блок `finally` для гарантированного закрытия потоков.
* `CustomStack.java` — реализация стека с собственным исключением `CustomEmptyStackException`.
* `ExceptionLogger.java` — класс для логирования ошибок в файл `exceptions.log`.

## Что изучено
1. **Обработка ошибок:** Использование блоков `try-catch-finally` для обеспечения устойчивости программы к некорректным входным данным.
2. **Создание пользовательских исключений:** Наследование от класса `Exception` для типизации специфических ошибок (например, пустого стека).
3. **Безопасная работа с ресурсами:** Гарантированное закрытие файловых потоков в блоке `finally`.
4. **Логирование:** Запись трассировки ошибок в файл для последующего анализа.



## Запуск
1. Скомпилируйте файлы: `javac *.java`
2. Запустите нужную программу: `java ИмяКласса` (например, `java FileCopy`)
