# Лабораторная работа №5: Регулярные выражения в Java

## Описание
Работа направлена на освоение регулярных выражений (`java.util.regex`) для комплексной обработки текстовой информации, валидации пользовательского ввода и контекстной замены текста. Программа объединяет 5 различных задач в рамках одного консольного приложения с интерактивным меню.

## Функциональность
* **Интерактивность:** Меню позволяет пользователю выбирать любое из 5 заданий для проверки, а в задании 5 поддерживается два режима работы (использование предустановленного текста или ввод собственного).
* **Поиск чисел (Задание 1):** Использование регулярного выражения `\d+\.\d+|\d+` для нахождения и вывода на экран всех целых и дробных чисел в тексте.
* **Валидация пароля (Задание 2):** Проверка строки на соответствие критериям безопасности с помощью блоков просмотра вперед (`^(?=.*[A-Z])(?=.*\d)[A-Za-z\d]{8,16}$`). Пароль должен быть длиной от 8 до 16 символов, содержать только латиницу и цифры, а также минимум одну заглавную букву и одну цифру.
* **Выделение заглавных букв (Задание 3):** Нахождение случаев, когда заглавная буква идет сразу после строчной без разделителей (например, в camelCase), с использованием свойств Unicode (`(?<=\\p{Ll})(?=\\p{Lu})`) и автоматическое выделение таких мест знаками `!!`.
* **Валидация IP-адреса (Задание 4):** Строгая проверка корректности ввода IPv4-адреса с контролем диапазонов каждого октета (от 0 до 255) через регулярное выражение.
* **Поиск слов по букве (Задание 5):** Использование регулярного выражения `\b[буква]\p{L}*\b` для корректного выделения слов, начинающихся с заданной пользователем буквы (с учетом границ слов `\b` и любых буквенных символов `\p{L}` в обоих регистрах).

## Основные классы
* `Lab5.java` — основной исполняемый файл, содержащий интерактивное меню, логику взаимодействия с пользователем, компиляцию шаблонов регулярных выражений и обработку возможных ошибок ввода.

## Как это работает
1. При запуске программа выводит меню со списком из 5 заданий и пункта выхода.
2. Пользователь выбирает номер задания и вводит необходимые демонстрационные данные (текст, пароль, IP-адрес или целевую букву).
3. Программа динамически обрабатывает входные данные, используя классы `Pattern` для компиляции выражений и `Matcher` для поиска совпадений или валидации.
4. В случае некорректного ввода (например, некорректный пункт меню или неверный формат данных) программа обрабатывает ошибку и выводит предупреждение.

## Инструкция по запуску
1. Скомпилируйте файл: `javac Lab5.java`
2. Запустите программу: `java Lab5`
