# Лабораторная работа №2: Наследование и полиморфизм

## Описание
Изучение объектно-ориентированного программирования (ООП):
* Создание абстрактного класса `Shape`.
* Реализация наследования классов `Sphere`, `Cylinder`, `Parallelepiped` от `Shape`.
* Использование переопределения методов (`@Override`) для вычисления объема и площади поверхности

## Запуск
1. Компиляция: `javac *.java`
2. Запуск: `java Main`
