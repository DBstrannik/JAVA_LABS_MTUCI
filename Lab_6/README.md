# Лабораторная работа №6: Коллекции, Дженерики и Потоки ввода-вывода

## Описание работы
Данная лабораторная работа направлена на закрепление навыков работы со структурами данных Java, обобщенным программированием (Generics) и потоками данных. Работа состоит из трех независимых задач.

## Структура проекта

### 1. Анализ частоты слов (Task1)
Программа считывает текст из файла `sample.txt`, подсчитывает количество вхождений каждого слова и выводит список из 10 самых часто встречающихся слов.
* **Технологии:** `HashMap`, `Scanner`, регулярные выражения (`Pattern`/`Matcher`).

### 2. Обобщенный стек (Task2)
Реализация стека (`Stack`), использующего дженерики (`<T>`). Позволяет хранить данные любого типа данных с проверкой на переполнение и опустошение.
* **Технологии:** `Generics`, обработка исключений (`RuntimeException`).

### 3. Система учета продаж (Task3)
Приложение для управления продажами, связывающее товары и их количество.
* **Технологии:** `Map`, объектная модель (`Product`), бизнес-логика подсчета общей стоимости и популярности товаров.

## Ключевые концепции
* **Коллекции:** Эффективное использование `Map` для быстрого поиска и доступа к данным.
* **Типобезопасность:** Использование обобщений для избежания приведения типов.
* **Обработка данных:** Чтение файлов и потоковая обработка текстовой информации.



## Инструкция по запуску
1. Убедитесь, что файл `sample.txt` находится в папке с проектом.
2. Скомпилируйте файлы: `javac *.java`
3. Запустите нужную задачу: `java Task1`, `java Task2` или `java Task3`.
