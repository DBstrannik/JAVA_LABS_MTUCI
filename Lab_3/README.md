# Лабораторная работа №3: Реализация Хэш-таблицы

## Описание работы
В рамках данной лабораторной работы изучены принципы построения хэш-таблиц. Была разработана собственная реализация хэш-таблицы на основе метода цепочек (связных списков) для разрешения коллизий.

## Структура проекта
* `Entry.java` — класс-контейнер, хранящий пару «ключ-значение».
* `HashTable.java` — основная реализация структуры данных. Содержит методы:
    * `put(K key, V value)` — добавление или обновление элемента.
    * `get(K key)` — получение значения по ключу.
    * `remove(K key)` — удаление элемента.
* `Student.java` — вспомогательный класс сущности для демонстрации работы с объектами.
* `Main.java` — точка входа, содержащая тесты для собственной `HashTable` и пример использования стандартной коллекции `java.util.HashMap`.

## Принцип работы
Хэш-таблица использует хэш-функцию для распределения ключей по индексам массива. При возникновении коллизий (когда разные ключи попадают в одну ячейку), используется `LinkedList` для хранения всех элементов с одинаковым хэшем.



## Запуск
1. Скомпилируйте все файлы: `javac *.java`
2. Запустите программу: `java Main`
